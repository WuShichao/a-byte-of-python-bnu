# 简明 Python 教程（北师大天文系版）

本书是北京师范大学天文系的“Python科学计算”课程教材，为低年级本科生介绍脚本编程语言Python 3 的基本概念，适用于没有任何编程基础的读者。本书是在 **Swaroop C H**所创作的《简明 Python 教程》（Byte of Python）一书的基础上修订而来。
《简明 Python 教程》是由 **Swaroop C H** 编写，旨在介绍 Python 语言的自由图书。2005 年，**沈洁元**将本书的 1.20 版引进中国，并完成了全本翻译，将本书的译名定为《简明 Python 教程》。
2017年，**漠伦**基于原书 4.0 版重新翻译，并沿用同样的译名。本书是在其 2018 年 2 月 4 日 4.08c 版的基础上进行调整和修改。
